Smiley

Meet our latest robot design, engineered for precision, speed, and versatility. Smiley represents the accumulation of months of iterative design, testing, and refinement. Sprinkled with a bit of smile.

  • Tech Stack: Roadrunner 1.0, MeepMeep, Pastuerized, OpenCV, and FTCLib
  • Drive Train: 3 deadwheel Mecanum wheel configuration for omnidirectional mobility and odometric localization
  • Mechanisms: String driven 5-stage Misumi linear slides with linkage system and uniclaw for both horizontal and vertical extension/manipulation. Double 4-stage belt driven viper slides for level 3 ascent
  • Autonomous: Roadrunner 1.0's 3 deadwheel localizer for precise localization and trajectory following. Customized for increased accuracy and optimizing cycle times

Technical Details

Programming

Our robot's software architecture is designed for reliability, precision, and adaptability.

  • Language: Java with FTC SDK and custom libraries
  • Navigation: Roadrunner trajectory generation and following with dead wheel odometry
  • Computer Vision: OpenCV for game element detection and field positioning
  • Control System: PID controllers for precision movement and stability

Mechanical Design

Our robot's mechanical systems are engineered for durability, efficiency, and performance.

  • Chassis: Custom-designed aluminum frame for optimal weight distribution
  • Drive System: Mecanum wheels for omnidirectional mobility with custom gearing
  • Extension Mechanism: Multi-stage linear slide system with custom pulley configuration
  • End Effector: Customized gripper designed for this season's game elements
