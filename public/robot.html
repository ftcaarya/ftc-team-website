MEET THE ROBOT

Engineered for precision, built for competition

CIRCUIT CRUSHER

Our competition robot for the 2024-2025 CENTERSTAGE season features a specialized intake system, a robust lifting mechanism, and precise pixel placement capability. Designed for efficiency and reliability, the robot can collect, transport, and place pixels with exceptional accuracy.

Weight

35 lbs

2.6 m/s

Languages

Java, Android Studio

KEY FEATURES

Dual Intake System

Our robot features a dual-sided intake system with rubber wheels for efficient pixel collection from both the playing field and human loading station. The design minimizes jamming and allows for quick collection of multiple pixels.

Telescoping Arm

A custom-designed telescoping arm provides extended vertical reach, allowing pixel placement at all backboard heights. The arm uses a cascading slide mechanism with a reliable chain-driven system for smooth extension and retraction.

Mecanum Drive

The robot uses a mecanum wheel drive system allowing omnidirectional movement for precise positioning. Powered by 4 high-torque motors, this drive system provides the agility needed to navigate the field efficiently during both autonomous and driver-controlled periods.

Advanced Control Hub

The robot brain consists of a Rev Robotics Control Hub running custom software developed by our team. This system coordinates motor control, sensor input, and autonomous routines, providing smooth operation during competitions.

Computer Vision

Using a webcam and custom TensorFlow models, our robot can identify and locate team prop positions during autonomous. The vision system also assists in precise alignment with the backboard for optimal pixel placement.

Custom End Effector

Our robot features a specialized gripper mechanism for secure pixel handling and precise placement. The gripper can hold multiple pixels simultaneously and includes a motorized release mechanism for controlled deployment.

PROGRAMMING

Software Architecture

Our robot's software is built using Java in Android Studio, following a modular architecture that separates core functionalities:

  • Hardware abstraction layer for seamless component integration
  • State machine for managing robot operational modes
  • PID controllers for precision movement
  • Computer vision system for game element detection
  • Autonomous routines with path planning capabilities

The team follows professional software development practices including version control with Git, code reviews, and comprehensive testing.

Autonomous Navigation

Our autonomous code uses dead reckoning and sensor fusion to navigate the field with precision. Multiple autonomous routines are programmed for different starting positions and game strategies.

Using TensorFlow Lite, our robot can identify team props and alignment markers. Custom trained models provide accurate detection even under variable lighting conditions.
